# S-400: レポート画面

## 概要

## 主要機能

### 1. レポート一覧表示

- 作成済みレポートをカード形式で一覧表示
- レポート名と簡単な説明を表示
- 各レポートに対する操作ボタン（表示、編集、削除）

### 2. レポート表示機能

- グラフやチャートによるデータの可視化
- サマリーテーブルによる数値の表示
- エクスポート機能（PDF、CSV 等）

### 3. レポート作成機能

- 新規レポートボタンから作成画面へ遷移
- カスタマイズ設定に基づいたレポート生成
- 各種集計・分析の実装

### 4. カスタマイズ設定機能

- レポート名の設定
- 表示項目の選択
- グラフタイプの選択（棒グラフ、円グラフ、折れ線グラフ等）
- 集計期間の設定（日次、週次、月次、年次）

### 5. レポート編集機能

- 既存レポートの設定変更
- レポートの更新・再生成

### 6. レポート削除機能

- 不要なレポートの削除

## 遷移先

- **ダッシュボード（S-200）**: ナビゲーションから遷移
- **データコンソール（S-300）**: ナビゲーションから遷移
