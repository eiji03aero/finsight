# S-110: ログインページ

## 概要

ユーザーがメールアドレスとパスワードでログインするための画面。

## 主要機能

### 1. ログインフォーム

- メールアドレス入力
- パスワード入力
- ログインボタン
- ログイン成功時はダッシュボードへ自動遷移

### 2. 新規登録ページへのリンク

- 新規ユーザー向けの LP ページへの誘導
- 「新規登録はこちら」リンクから LP ページへ遷移

## 遷移先

- **ダッシュボード（S-200）**: ログイン成功時
- **LP ページ（S-100）**: 「新規登録はこちら」リンクから
