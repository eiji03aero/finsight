## 主要機能

### 1. ヘッダー

- ロゴ表示（テキスト形式: "Finsight"）
- 現在のページ名表示
- ユーザー情報・メニュー

### 2. ナビゲーション

- ダッシュボードへのリンク
- データ一覧画面へのリンク
- レポート画面へのリンク

### 3. メインコンテンツエリア

- 現在は空（今後、サマリー情報や重要な指標などを表示予定）

## 遷移先

- **データコンソール（S-300）**: ナビゲーションから遷移
- **レポート画面（S-400）**: ナビゲーションから遷移
