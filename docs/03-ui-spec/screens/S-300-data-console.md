# S-300: データコンソール

## 概要

投入したデータを一覧で確認できる画面。絞り込み、データ登録・編集、CSV アップロード、マスタ管理が可能。

## 主要機能

### 1. データ一覧表示

- データをテーブル形式で表示
- 日付、項目、金額、操作ボタンなどの列を表示
- ページネーション機能

### 2. 絞り込み機能

- 期間による絞り込み（開始日〜終了日）
- カテゴリによる絞り込み
- 検索機能

### 3. データ手動登録機能

- 新規登録ボタンからフォームを表示
- 日付、項目、金額、カテゴリ、メモなどを入力
- データの保存

### 4. データ編集機能

- 各行の編集ボタンから編集フォームを表示
- 既存データの修正・更新
- データの削除

### 5. CSV 取込機能

- CSV ファイルのアップロード
- ファイル選択またはドラッグ&ドロップ
- 一括データ取込

### 6. マスタ管理機能

- カテゴリマスタの管理（追加・編集・削除）
- 項目マスタの管理（追加・編集・削除）
- マスタデータの一覧表示

## 遷移先

- **ダッシュボード（S-200）**: ナビゲーションから遷移
- **レポート画面（S-400）**: ナビゲーションから遷移
