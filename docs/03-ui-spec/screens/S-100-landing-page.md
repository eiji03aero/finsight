# S-100: LPページ（ランディングページ）

## 概要

## 主要機能

### 1. ユーザー登録フォーム

- メールアドレス入力
- パスワード入力
- パスワード確認入力
- 新規登録ボタン
- 登録成功時はダッシュボードへ自動遷移

### 2. ログインページへのリンク

- 既存ユーザー向けのログインページへの誘導
- 「ログインはこちら」リンクからログインページへ遷移

### 3. サービス紹介セクション

- キャッチコピー表示: 「家計データを丸裸に」
- サービスの特徴・メリットの紹介
- 主要機能のハイライト

## 遷移先

- **ダッシュボード（S-200）**: ユーザー登録成功時
- **ログインページ（S-110）**: 「ログインはこちら」リンクから
