# 用語集

本ドキュメントは FinSight プロジェクトで使用される用語を定義します。

| 用語                     | 定義                                                                                           |
| ------------------------ | ---------------------------------------------------------------------------------------------- |
| ワークスペース           | 家計簿データを管理する単位。複数のユーザーが所属可能で、データはワークスペース単位で隔離される |
| ワークスペースオーナー   | ワークスペースを作成したユーザー。メンバー管理などの管理権限を持つ                             |
| ワークスペースメンバー   | ワークスペースに招待され所属しているユーザー                                                   |
| ユーザー                 | システムの利用者。1 人のユーザーは複数のワークスペースに所属できる                             |
| 収支データ               | ワークスペースに入力または登録される支出・収入の明細データ                                     |
| マスタ                   | カテゴリなど、収支データを分類・管理するための基準データ                                       |
| テンプレート             | CSV ファイルの列とシステム内のデータ項目のマッピング設定                                       |
| カテゴリ                 | 支出・収入の分類（例：外食費、Grocery、給与など）                                              |
| 繰り返しトランザクション | 定期的に発生する収支データ（例：月額固定費、定期収入など）期間指定可能                         |
| 変動費                   | その都度発生する支出                                                                           |
| ドリルダウン             | 集計データから詳細な内訳データへと掘り下げて表示する機能                                       |
