# プロジェクト概要

## プロジェクトの基本情報

- **プロジェクト名**: FinSight
- **システム種別**: Web アプリケーション（新規開発）
- **対象ユーザー**: 家計簿を管理する個人ユーザー、または複数人で家計を共有管理するグループ

## 背景と目的

### 背景

現在、多くのユーザーがスプレッドシートを用いて家計簿管理を行っているが、以下のような課題が存在する：

- データのアップロードが煩雑である
- シートの数が増えると管理が困難になる
- シート間の行き来が面倒である
- 運用性や拡張性に限界がある

### 目的

FinSight は、これらの課題を解決し、以下の価値をユーザーに提供することを目的とする：

- 家計簿の簡単な管理を実現する
- 直感的な操作でマスタデータを的確に作成できる
- 視覚化や集計機能により、家計改善の示唆を提供する
- より効率的で拡張性の高い家計管理環境を提供する
- ワークスペース機能により、個人利用から複数人での家計共有まで柔軟に対応する

## システム構成

### 技術スタック

#### フロントエンド

- React
- TanStack Start

#### バックエンド

- Go
- Gin（Web フレームワーク）
- Ent（ORM フレームワーク）

#### データベース

- PostgreSQL

#### 外部連携

- 現時点では外部サービスとの連携は不要

### システムアーキテクチャ概要

```
[ユーザー]
    ↓
[Webブラウザ（React + TanStack Start）]
    ↓ HTTP/HTTPS
[APIサーバー（Go + Gin）]
    ↓
[データアクセス層（Ent）]
    ↓
[PostgreSQL データベース]
```

## 今後の拡張可能性

以下の機能は初期リリースには含まれないが、将来的な実装を検討する可能性がある：

- 予算設定機能と予算超過アラート
- マスタの自動紐付けルール（機械学習的なアプローチ）
- データのエクスポート機能（CSV、PDF 等）
- スマートフォン対応（レスポンシブデザイン、PWA 化）
- 複数デバイスでのリアルタイム同期
- グラフの種類追加（円グラフ、棒グラフ等）
- レシート画像からの OCR 機能
- 銀行・クレジットカードとの自動連携
- 家計改善の自動提案機能
